# @grace-studio/graceful-figma

## Installation

```
npm i -g @grace-studio/graceful-figma
```

## Usage

```
graceful-figma react-icons \
--key <project-key> \
--page <page-name> \
--section <section-name> \
--out <./output/dir>
```

## Optional .env

The access token can be placed in the `.env` file

```
#.env
FIGMA_ACCESS_TOKEN=your-secret-access-token
```
